# MokaData 财务数据分析平台 - 离线版

## 快速开始

### 系统要求
- Windows 7 / 10 / 11
- Node.js v18 或以上版本（已确认已安装）

### 启动方法

**双击 `start.bat` 即可启动**

首次运行会自动安装依赖（约 1-2 分钟），之后每次启动只需几秒。

启动成功后，浏览器会自动打开：
- 本机访问：http://localhost:3737
- 局域网访问：http://[本机IP]:3737（其他圈内电脑也可访问）

### 停止服务

在命令行窗口按 `Ctrl+C` 即可停止服务。

## 功能说明

### 参数配置（核心功能）
支持以下 7 类参数的手动录入、Excel 批量导入和模板下载：

| 配置类型 | 字段 |
|---------|------|
| SKU 配置 | SKU编码、中文名称、英文名称、SKU类别 |
| 汇率配置 | 月份、国家、币种、对人民币汇率 |
| 关税配置 | SKU编码、中英文品名、出口/进口HSCODE、关税税率、报关价格 |
| 头程(国家+SKU) | SKU编码、运输目的地、运输方式、头程单价 |
| 头程(国家+品类) | 品类名称、运输目的地、运输方式、头程单价 |
| 头程(品类兜底) | 品类名称、运输方式、头程单价 |
| 尾程配置 | 文件来源、物流商名称、国家名称、仓库名称 |

**头程费用匹配顺序**：① 国家+SKU → ② 国家+品类 → ③ 仅品类（兜底）

### Excel 导入
1. 点击「下载模板」获取标准格式
2. 按模板填写数据
3. 点击「上传 Excel」选择文件
4. 选择导入模式：
   - **增量追加**：保留现有数据，新增不重复条目
   - **全量覆盖**：清空后全量导入

### 文件管理
上传 Excel/CSV 数据文件，支持分组和标签管理。

## 数据存储

所有数据存储在 `data/mokadata.db`（SQLite 数据库文件），无需外网连接。

**备份方法**：直接复制 `data/` 文件夹即可完整备份所有数据。

## 常见问题

**Q: 启动后浏览器没有自动打开？**
A: 手动在浏览器地址栏输入 `http://localhost:3737`

**Q: 端口 3737 被占用？**
A: 在 start.bat 中修改 `set PORT=3737` 为其他端口号，如 `set PORT=8080`

**Q: 如何让局域网内其他电脑访问？**
A: 查看本机 IP（运行 `ipconfig`），其他电脑访问 `http://[你的IP]:3737`
